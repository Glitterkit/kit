# 🌌 无界 (WuJie) - 万界追踪工作台

> 你的个人 AI 助手，像贾维斯一样管理一切

## ✨ 功能特性

### 五大核心模块

1. **📊 总览仪表盘**
   - 实时统计数据展示
   - 最近活动追踪
   - 一目了然的工作台概览

2. **🏠 生活追踪**
   - 健康管理（运动、饮食）
   - 学习计划
   - 家庭事务
   - 个人成长记录

3. **💼 工作追踪**
   - 项目管理
   - 会议记录
   - 报告追踪
   - 工作任务管理

4. **💻 开发追踪**
   - 项目开发进度
   - 技术学习路线
   - 代码任务管理
   - 技能提升计划

5. **👁️ 查看追踪**
   - 文章阅读记录
   - 视频观看进度
   - 链接收藏管理
   - 内容消费追踪

6. **🧠 知识库**
   - 技术笔记整理
   - 知识标签分类
   - 快速检索查询
   - 经验沉淀积累

## 🎨 设计特点

- **深色主题**: 护眼科技感 UI 设计
- **响应式布局**: 支持桌面端和移动端
- **流畅动画**: 优雅的交互体验
- **智能搜索**: 全局内容检索

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```

## 📦 技术栈

- **框架**: React 18 + Vite
- **UI 组件**: Lucide React Icons
- **样式**: 原生 CSS + CSS Variables
- **状态管理**: React Hooks

## 🎯 使用指南

### 添加新记录
1. 点击任意模块的「添加」按钮
2. 填写标题、选择分类、添加备注
3. 点击确认保存

### 更新状态
- 在卡片底部使用状态下拉菜单
- 实时更新记录状态

### 搜索内容
- 使用顶部搜索栏
- 支持标题和备注全文搜索

### 导航切换
- 左侧边栏切换不同模块
- 移动端可折叠侧边栏
